## Get


| Option                     | Description                              |                 |
| -------------------------- | ---------------------------------------- | --------------- |
| ``--from``                 | Specify the vault.                       | Default: None   |


### Example

```
$ passgen get mail_password --from business_vault
Enter the password for vault: *********
Are you sure you want to reveal your password here? [y\N]: y
mail_password --> u82hHnmaD
```
